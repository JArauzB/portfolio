# Prepaid SIM Card Booking System

**Category:** Web Application & E-commerce  
**Tools:** PHP, MySQL, Payment Integration, Booking Management

Developed a comprehensive online booking and management system for prepaid SIM cards. The platform enables customers to browse plans, make reservations, process payments, and manage activations. Includes admin dashboard for inventory tracking, order management, and customer analytics.
