# PlantCare IoT Monitoring System

**Category:** IoT & Embedded Systems  
**Tools:** Arduino, Sensors, Node.js, Real-time Data

Designed and built an IoT-based plant monitoring system that tracks soil moisture, temperature, humidity, and light levels. The system provides real-time data visualization and automated alerts to optimize plant care and reduce water waste through intelligent irrigation control.
